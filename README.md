## LINK HEROKU

https://notes-api-backend2021.herokuapp.com/api/notes

## API endpoints

`GET api/notes`

Retorna todas las notas.

---

`GET /api/notes/{id}`

Devuelve los detalles de la nota con la id correspondiente.

---

`POST /api/notes`

Agrega una nota.

---

`PUT /api/notes/{id}`

Actualiza los datos de una nota.

---

`DELETE /api/notes/{id}`

Elimina una nota.
